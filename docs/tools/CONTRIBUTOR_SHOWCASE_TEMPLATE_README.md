# Contributor Showcase Template README

## Important note

This project should only display real contributors or clearly marked placeholder profiles.
Do not add fake identities pretending to be real contributors.

## Why this exists

You wanted a contributor profile/picture section.
This template gives a safe way to add team recognition without misleading repository visitors.

## How to use in this project

Create a section in your docs with cards like:

- Name
- Role
- Contribution area
- Avatar image path
- Social/profile link

If a profile is not real yet, mark it as `Template` or `Placeholder`.

## Example markdown card

```md
### Jane Doe
- Role: Platform Engineer
- Focus: Kubernetes and CI/CD
- Avatar: docs/assets/contributors/jane-doe.png
- Profile: https://github.com/janedoe
```

## Recommended folder for images

- `docs/assets/contributors/`

Use only images you have rights to use.
